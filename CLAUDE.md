## Project Overview

**ratex** is an R package that provides LaTeX colors for R data visualizations.

- GitHub: https://github.com/jmaasch/ratex

## R Package Development Commands

```r
# Install dependencies
install.packages("devtools")

# Load package in development
devtools::load_all(".")

# Run all tests
devtools::test()

# Run a single test file
testthat::test_file("tests/testthat/test-<name>.R")

# Check the package (CRAN-style)
devtools::check()

# Build documentation
devtools::document()

# Install locally
devtools::install()
```

## Standard R Package Structure (expected)

As the package grows, it will follow the standard R package layout:

- `R/` — R source files (exported functions and internals)
- `tests/testthat/` — unit tests using the `testthat` framework
- `DESCRIPTION` — package metadata, dependencies, version
- `NAMESPACE` — exported symbols (auto-generated by `devtools::document()`)
- `man/` — Rd documentation files (auto-generated from roxygen2 comments)
- `data/` — bundled datasets (if any)

## Development Notes

- Documentation is written as roxygen2 comments (`#'`) above each function.
- Run `devtools::document()` after changing roxygen2 comments to regenerate `NAMESPACE` and `man/`.
- The package is intended to expose LaTeX-standard color palettes for use with `ggplot2` or base R graphics.
