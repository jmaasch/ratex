ratex

LaTeX colors for R data visualizations.

R Package MIT License Open Source
devtools::install_github("jmaasch/ratex")

Overview

ratex brings the full LaTeX color vocabulary into R, making it easy to use publication-quality colors in ggplot2 figures, base R graphics, and any other visualization workflow.

Whether you are typesetting a document in LaTeX and want your plots to match, or simply want a curated palette of named colors with well-known hex values, ratex gives you a consistent, searchable interface to hundreds of colors drawn from the dvipsnames, X11, and latexcolor.com color sets.

Installation

Install the development version from GitHub:

# install.packages("devtools")
devtools::install_github("jmaasch/ratex")

Then load the package:

library(ratex)

Usage

Look up a color by name and get its hex value:

ratex("MidnightBlue")
#> [1] "#006795"

Use ratex colors directly inside a ggplot2 call:

library(ggplot2)

ggplot(mtcars, aes(wt, mpg)) +
  geom_point(color = ratex("BurntOrange"), size = 3) +
  theme_minimal()

Pass a vector of names to build a custom palette for use with scale_color_manual() or scale_fill_manual(): 

my_palette <- ratex(c("NavyBlue", "BurntOrange", "ForestGreen", "Plum"))

ggplot(iris, aes(Sepal.Length, Sepal.Width, color = Species)) +
  geom_point(size = 2) +
  scale_color_manual(values = my_palette) +
  theme_minimal()

Functions

Function Description
ratex(name) Return the hex code for one or more named LaTeX colors.
ratex_search(pattern) Search available color names by substring or regex.
ratex_palette(n) Return a balanced palette of n colors.
ratex_show(name) Plot a visual swatch for one or more colors.
